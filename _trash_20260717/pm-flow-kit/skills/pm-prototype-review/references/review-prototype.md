# Prototype Review Artifact

Use a prototype review artifact when reviewers need to compare UI, rules, fields, and acceptance criteria in one place.

## Required Sections

- Review scope.
- Referenced prototype files.
- Scenario coverage.
- UI-to-rule mapping.
- Field and data checks.
- Acceptance criteria checks.
- Findings.
- Open questions.
- Decisions.
