pakco.html · 网页模式

自然流动的网页形态

没有分页，没有边界。内容随浏览器宽度自适应排版，手机上下滑动阅读。

Features

三个核心特性

无边界

内容自然流动，不再被锁在 16:9 的格子里。长段文字、多张卡片，想放多少放多少。

自适应

4 档响应式断点：手机全宽、平板双列、桌面 1080px、2K 宽屏 1400px。grid 自动降列。

零依赖

不需要 runtime.js，不需要键盘导航。纯 CSS 驱动，浏览器原生滚动。

Result

100%

浏览器原生体验

上下滑就行，不用学任何操作

Comparison

Deck vs 网页

Deck 模式

适合演讲、分享、Keynote 场景。全屏 16:9，键盘/手势翻页，有演讲者视图。

网页模式

适合阅读、传播、存档。自然滚动，手机友好，可以发链接给任何人直接看。

Usage

怎么用

把 <body class="webpage"> 加上，用 .page 包裹内容，用 <section> 分段。不需要 .deck、.slide、runtime.js。

主题照样切换 — 换 theme-link 的 href 就行。所有 token（颜色、字体、圆角）通用。
