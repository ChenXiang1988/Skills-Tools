# guizang-swiss

瑞士国际主义风格，by 歸藏。WebGL 极细网格 + 点阵背景 + 全程无衬线 + 极致字号对比。
支持 4 套主题色：克莱因蓝 IKB（默认）/ 柠檬黄 / 柠檬绿 / 安全橙。

**Use when:** 科技产品、数据汇报、设计/工程领域分享、年度总结。
**Feel:** Massimo Vignelli + Helvetica Forever。
**Theme switch:** `?theme=ikb|lemon|green|orange`
**Source:** https://github.com/op7418/guizang-ppt-skill (by 歸藏)
