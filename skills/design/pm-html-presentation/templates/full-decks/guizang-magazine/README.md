# guizang-magazine

电子杂志 × 电子墨水风格，by 歸藏。WebGL 流体/色散背景 + 衬线大标题 + 横向翻页。
支持 5 套主题色：墨水经典（默认）/ 靛蓝瓷 / 森林墨 / 牛皮纸 / 沙丘。

**Use when:** 人文分享、行业观察、商业发布、需要杂志感的演讲。
**Feel:** Monocle 杂志 + 代码美学。
**Theme switch:** `?theme=ink|indigo|forest|kraft|dune`
**Source:** https://github.com/op7418/guizang-ppt-skill (by 歸藏)
