# Context Confirmation

Context confirmation prevents agents from pretending they used a knowledge base.

## Required Answers

- Is an external knowledge base used?
- What is the context root?
- What manifest or index was read?
- What intake file was used?
- Which sources are applicable?
- Which sources are not applicable?
- What MUST and FORBIDDEN rules apply?
- Are there conflicts?
- Which sources must be cited or avoided in later stages?

Do not advance requirement analysis until context confirmation is complete.
