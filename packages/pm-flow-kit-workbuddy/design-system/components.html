业务状态色板（仓库专用语义）

非 antd 原生，是云仓业务语义约定。表格状态列 / Tag / 进度统一引用，勿与品牌蓝混淆。

待收货
--blue-1 / 主色文字
收货中
#FFFBE6 / warning 文字
已上架
#F6FFED / success 文字
异常
#FFF1F0 / error 文字
主操作
--color-primary
/* 业务状态语义 */
待收货  --blue-1 底 + 主色文字
收货中  #FFFBE6 底 + warning 文字
已上架  #F6FFED 底 + success 文字
异常    #FFF1F0 底 + error 文字
